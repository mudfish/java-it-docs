# 探测端口的几种方式

  1. 常用telnet 118.10.6.128 88方式测试远程主机端口是否打开。

  2. nmap ip -p port 测试端口

nmap ip 显示全部打开的端口，根据显示close/open确定端口是否打开。

  1. nc -v host port

端口未打开返回状态为非0
